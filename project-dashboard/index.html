F / H 狀態看板

F 硬碟

冷凍穩定基準
  • 定位：正式穩定基準與救生艇。
  • 目前保持 `v2026.07.01-18`，星期二前不部署實驗功能。
  • 不切 feature、不掃 5TB、不修改原始資料。
  • 如果 H 實驗失敗，F 仍作為可退回的實體基準。

H 硬碟

受控實驗硬碟
  • 定位：唯一實驗、開發與驗收硬碟。
  • 目前功能集中在動態索引、Watcher、行政 POC、H explorer 與 H-wide explorer。
  • 危險操作只在明確安全 sandbox 測試；正式資料不做破壞性操作。
  • 刪除第一階段只移到 H 安全回收區，不永久刪除。

Git / GitHub / BAT / SQLite / Watcher / H Explorer 的角色

Git

保存程式修改歷史、commit、分支與可回復點。Git 不保存原始 5TB 資料。

GitHub

保存共用程式、文件、安全截圖與驗收相簿。Private repo 不是 F/H 原始資料備份。

BAT

一般使用者入口。未來正式流程應是點 BAT 後檢查程式更新、資料索引與啟動服務。

SQLite

每顆硬碟自己的本機索引。GitHub 保存索引程式，不保存正式 SQLite 檔案。

Watcher

服務運行期間監聽受控資料夾異動，經 debounce 後更新索引。事件遺失時仍需啟動核對。

H Explorer

H 實驗版檔案瀏覽器。H-wide 模式可瀏覽 H 主要資料區，系統保護區禁止 mutation。
